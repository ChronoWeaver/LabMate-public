支持页面

湿实验助手（WetLab Pro）是一款面向湿实验场景的本地优先效率工具，用于管理实验模板、实验安排、补充记录，以及完成常见实验计算与实用工具操作。

联系方式

支持邮箱：guozhongfang2020@icloud.com

问题反馈：GitHub Issues

常见支持主题

  • 安装、启动与兼容性问题
  • 实验模板创建、编辑、分类与复制
  • 实验安排、提醒与通知行为
  • 补充记录中的文字与图片管理
  • 数据导出、恢复与迁移
  • iCloud 同步开启、关闭和同步行为
  • 配液、浓度、分子量、RPM/RCF 等计算结果问题
  • 多任务计时器、盲点计数器和历史记录问题

反馈问题时建议提供的信息

  • 设备型号
  • iOS 版本
  • App 版本
  • 问题出现前的操作步骤
  • 你的预期结果
  • 实际结果
  • 如问题与界面显示有关，可附截图

使用说明

  • 本应用无需登录账号即可使用主要功能
  • 通知权限仅用于实验提醒
  • 相机与相册权限仅用于实验补充记录图片相关功能
  • iCloud 同步为可选功能，由用户主动开启

Support

WetLab Pro is a local-first productivity app for wet-lab workflows. It helps manage experiment templates, schedules, supplemental logs, and practical lab utilities.

Contact

Support email: guozhongfang2020@icloud.com

Issue tracker: GitHub Issues

Common Topics

  • Installation, launch, and compatibility issues
  • Experiment template creation, editing, categories, and duplication
  • Experiment schedules, reminders, and notification behavior
  • Text and image handling in supplemental logs
  • Data export, restore, and migration
  • iCloud sync enable, disable, and sync behavior
  • Questions about dilution, molarity, molecular weight, and RPM/RCF calculations
  • Multi-timer, blind counter, and history issues

What To Include In A Report

  • Device model
  • iOS version
  • App version
  • Steps to reproduce
  • Expected result
  • Actual result
  • A screenshot when the issue is UI-related

Usage Notes

  • The app does not require account sign-in for core features
  • Notification permission is only used for reminders
  • Camera and Photos permissions are only used for supplemental log images
  • iCloud sync is optional and user-controlled
